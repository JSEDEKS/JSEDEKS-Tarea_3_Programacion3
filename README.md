# JSEDEKS-Tarea_3_Programacion3
Repositorio académico para la Tarea #3 de Programación 3. Incluye un proyecto CRUD y la aplicación completa de Git Flow: ramas main, developer y qa, más 5 ramas feature/hotfix. Cada rama genera Pull Requests hacia las 3 ramas principales, completando un total de 15 PR cerrados como evidencia del flujo de trabajo profesional.
