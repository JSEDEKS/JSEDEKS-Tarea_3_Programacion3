# Repositorio académico para la Tarea #3 de Programación 3.

Incluye un proyecto CRUD y la aplicación completa de Git Flow: ramas main, developer y qa, más 6 ramas feature/hotfix.
Cada rama genera Pull Requests hacia las 3 ramas principales, completando un total de 18 PR cerrados como evidencia del flujo de trabajo profesional.




# Instrucciones para ejecutar el programa
# iniciar
npm i

# ejecutar 

npm start
